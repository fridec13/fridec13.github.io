SSAFY 12기 공통 프로젝트로 개발한 블록체인 기반 반려동물 의료정보 관리 시스템입니다. 프라이빗 이더리움 네트워크(Besu)를 활용해 가스비 없이 트랜잭션을 처리하고, 보호자가 진료 기록을 블록체인에 저장·조회하며 병원별 접근 권한을 직접 제어하는 구조를 구현했습니다. 블록체인 및 프론트엔드를 담당했습니다.

서비스가 필요한 이유

  • 병원을 변경할 때마다 반려동물의 과거 진료 내용을 직접 설명해야 하는 불편함이 존재합니다.
  • 보호자가 예방접종 내역이나 병력 정보를 잊어버리면 정확한 진료가 어려워질 수 있습니다.
  • KKUKKKUK은 기록을 분실 걱정 없이 블록체인에 보관하고, 필요 시에만 병원과 안전하게 공유할 수 있습니다.

의료 기록 조회

핵심 설계 — 보호자 중심 접근 제어

기존 의료 시스템에서 진료 기록의 소유권은 병원에 있었습니다. 이 프로젝트는 스마트컨트랙트를 통해 보호자가 기록의 소유권을 갖고, 병원별로 조회 권한을 부여하거나 회수하는 구조로 설계했습니다. 

// PetMedical.sol (핵심 로직)
mapping(address => mapping(address => bool)) private permissions;
// permissions[보호자][병원] = 조회 권한 여부

function grantAccess(address hospital) external {
    permissions[msg.sender][hospital] = true;
    emit AccessGranted(msg.sender, hospital);
}

function revokeAccess(address hospital) external {
    permissions[msg.sender][hospital] = false;
    emit AccessRevoked(msg.sender, hospital);
}

function getMedicalRecord(address owner, uint256 petId)
    external view returns (MedicalRecord memory)
{
    require(
        msg.sender == owner || permissions[owner][msg.sender],
        "Access denied"
    );
    return records[owner][petId];
}

담당 역할

스마트컨트랙트 설계 및 배포

Solidity로 반려동물 의료 기록 저장·조회·권한 관리 로직을 구현하고 Remix IDE를 통해 프라이빗 이더리움 네트워크에 배포했습니다. 반복 테스트를 위해 배포 스크립트를 Hardhat으로 자동화하고, 컨트랙트 ABI를 프론트엔드에서 직접 임포트하는 방식으로 연동 구조를 단순화했습니다.

MetaMask 연동 DApp 구현

React에서 ethers.js를 사용해 MetaMask 지갑 연결, 트랜잭션 서명, 컨트랙트 함수 호출을 구현했습니다. 지갑 미연결 상태, 네트워크 불일치, 트랜잭션 거부 시 사용자 친화적인 피드백을 제공하도록 예외 처리를 작성했습니다.

트랜잭션 성능 최적화

진료 기록 조회가 잦은 병원 측 케이스에서 매번 call()을 트랜잭션으로 처리하는 것이 불필요한 가스 소모로 이어진다는 점을 발견했습니다. 읽기 전용 함수에 view 키워드를 적용하고 로컬에서 직접 호출하도록 변경하여 조회 호출이 체인에 기록되지 않고 즉시 반환되도록 개선했습니다.
