SSAFY 12기 관통 프로젝트로 개발하여 최우수상을 수상한 AI 기반 음성 제어 로봇 공정 자동화 시스템입니다. 4DOF Dobot Magician 로봇팔, RealSense 카메라, 라즈베리파이 컨베이어를 ROS2로 통합하고, STT + ChatGPT 파이프라인으로 음성 명령만으로 전체 공정을 제어합니다. RoboDK 디지털 트윈을 통해 실시간 상태 동기화도 구현했습니다.

음성 제어 — STT + ChatGPT 파이프라인

ChatGPT는 사용자 발화를 공정 명령과 일상 대화로 구분합니다. 공정 명령은 JSON 구조체로 파싱되어 ROS2 토픽으로 발행되고, 일상 대화는 TTS 응답으로 돌아와 자연스러운 인터랙션을 제공합니다.

픽앤플레이스 노드 + ChatGPT 연동

ChatGPT로부터 파싱된 명령 구조체를 구독하는 ROS2 노드를 구현하고, Dobot SDK API를 호출하여 순차 동작(이동 → 흡착 → 적재)을 수행하는 픽앤플레이스 파이프라인을 작성했습니다. 음성으로 "3번 부품 좌측에 놓아줘"라고 말하면 GPT가 {"action": "pick_and_place", "part_id": 3, "position": "left"} 형태로 파싱해 노드에 전달합니다. 

# pick_and_place_node.py
def command_callback(self, msg: String):
    cmd = json.loads(msg.data)
    if cmd["action"] == "pick_and_place":
        part_pos = self.detect_part(cmd["part_id"])   # 카메라로 부품 위치 파악
        self.robot.move_to(part_pos)
        self.robot.suction_on()
        self.robot.move_to(PLACE_POSITIONS[cmd["position"]])
        self.robot.suction_off()
        self.get_logger().info(f"Part {cmd['part_id']} placed at {cmd['position']}")

라즈베리파이 컨베이어 연동

라즈베리파이에서 모터 드라이버를 제어하는 ROS2 노드를 구현했습니다. YOLO 검출 결과를 구독하여 불량 부품이 배출 구간에 도달하면 컨베이어를 정지하고 서보모터로 불량품을 분리하는 로직을 작성했습니다.

발표 및 발표자료

팀 발표를 담당하여 시스템 파이프라인과 데모 시연 흐름을 정리한 발표자료를 제작했습니다. 관통 프로젝트 최우수상 수상.

기술적 도전 — 실시간 STT 지연 최소화

RTZR STT API는 발화 종료 시점부터 결과 반환까지 평균 1~2초의 지연이 있어 로봇 동작 시작 전에 긴 정지 구간이 생기는 문제가 있었습니다. STT 요청을 별도 비동기 스레드에서 처리하고, ChatGPT 파싱 단계를 최대한 짧은 프롬프트로 최적화하여 전체 음성→동작 시작 지연을 약 3~4초 수준으로 유지했습니다.
